# Простой блог на FastAPI

Минималистичный блог с API на FastAPI для начинающих.

## Функции:
- Регистрация и вход пользователей
- Создание, чтение, обновление и удаление статей
- Комментарии к статьям
- Теги для статей
- Поиск по статьям
- Пагинация
